NEW

CSS Grid로 만드는 깔끔한 구조

Grid를 사용하면 웹 페이지를 행과 열 기준으로 나눌 수 있어 복잡한 레이아웃도 훨씬 쉽게 구성할 수 있습니다.
이번 매거진의 전체 구조도 3단 Grid로 배치했습니다. grid 사용에 있어서 교수님이 주신 연습 사이트가 도움이 되었습니다. 매거진 독자 여러분들도 grid garden을 이용해 배워보세요!

HOT

게임처럼 생동감 있는 웹페이지

웹페이지도 게임처럼 시선을 끄는 요소가 중요합니다. 상단 속보 텍스트에는 흐르는 효과를, 기사 뱃지에는 둥둥 떠오르는 애니메이션을 넣어 더 생동감 있게 만들었습니다.
나중에 배움을 더 쌓아서 다양한 애니메이션을 사용해보는 시간도 가져보고 싶습니다.

TIP

반응형 웹의 중요성

PC 화면에서는 다단 레이아웃으로 보이지만, 모바일처럼 화면이 좁아지면 1단으로 바뀌어야 읽기 편합니다.
그래서 media query를 사용해 반응형 구조도 함께 구현했습니다. 처음 이코드를 작성할때 이해가 안갔지만 결국 성공했습니다.

BEST

CSS 변수로 디자인 관리하기

색상 값을 변수로 정리하면 전체 테마를 쉽게 바꿀 수 있습니다. 배경색, 강조색, 테두리색을 var로 관리해 코드의 통일성을 좋게 만들어보았습니다.
계속해서 나오는 다양한 기사 썸네일은 교수님께서 사용하신 Unsplash.com 사이트에서 마음에 드는 사진을 골라 적용해보았습니다.
